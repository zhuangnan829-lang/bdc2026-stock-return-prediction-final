# 正式模型对比表

## 对比口径

- 当前正式默认特征集：`base_alpha_v3_rs_crowding_mini4`
- 当前正式默认排序策略：`risk_adjusted`
- 当前正式默认权重策略：`pred`
- 当前正式默认 `top_k`：`5`
- `NDCG@K`：按每日预测排序前 K 只股票的真实收益计算 DCG，再用真实收益理想排序归一化后求均值。
- `HitRate@K`：按每日预测 Top-K 与真实收益 Top-K 的股票重合比例求均值。
